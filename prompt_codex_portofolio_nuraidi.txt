Buat sebuah website personal portfolio bernama “Portofolio Nuraidi” menggunakan React.js, Vite, JavaScript, JSX/HTML, dan CSS. Website harus menggunakan bahasa Indonesia, responsif, cepat, mudah dipelihara, serta siap disimpan di GitHub dan di-deploy ke Vercel.

KONTEKS PROFIL

Nuraidi bukan programmer profesional. Pengembangan website dan aplikasi merupakan hobi, minat pribadi, serta kemampuan digital pendukung. Jangan menampilkan Nuraidi sebagai Full-Stack Developer, Front-End Developer, Software Engineer, atau programmer penuh.
